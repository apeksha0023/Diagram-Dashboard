# ⚡ Simple Diagram Dashboard

A modern **React Dashboard** that allows users to upload circuit diagram images and view detected electronic components in a sidebar panel.

This project demonstrates **React fundamentals, UI design, and clean component architecture**.

---

## 📌 Features

### 📤 Upload Diagram
- Upload PNG or JPG images
- Display uploaded file name
- Replace the uploaded image

### 🔍 Diagram Viewer
- Zoom In
- Zoom Out
- Reset View
- Smooth image scaling

### 📋 Components List Panel
- Displays detected electronic components
- Highlights selected component
- Interactive sidebar

### 📊 Dashboard Layout
- Upload section at the top
- Diagram viewer on the left
- Components list on the right

### 📱 Responsive Design
- Desktop layout
- Tablet support
- Mobile stacked layout

### 🎨 Modern UI
- Animated gradient background
- Clean dashboard card layout
- Smooth hover animations

---

## 🛠 Tech Stack

- React.js
- React Hooks (useState, useEffect)
- CSS3
- Flexbox
- Mock API

## 🚀 Installation & Setup

### 1️⃣ Clone Repository

```bash
git clone https://github.com/your-username/diagram-dashboard.git
```

### 2️⃣ Navigate to Project Folder

```bash
cd diagram-dashboard
```

### 3️⃣ Install Dependencies

```bash
npm install
```

### 4️⃣ Run Development Server

```bash
npm run dev
```

Application will run on:

```
http://localhost:5173
```
